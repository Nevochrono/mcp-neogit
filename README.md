## 🏁 Getting Started

Follow these steps to get your development environment set up:

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd mcp-neogit
   ```
2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the application**
   ```bash
   python main.py
   ```
